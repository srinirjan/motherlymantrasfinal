# Quick Setup Guide

## Simple Setup - No Configuration Needed!

This app uses simple email-based authentication with no external services required.

## Run the App

```bash
npx expo start
```

Then press:
- `i` for iOS simulator
- `a` for Android emulator  
- `w` for web browser

## How Authentication Works

1. **New Users**: Enter your email and name to create an account
2. **Returning Users**: Just enter your email to sign in
3. **No Passwords**: Simple, secure email-based authentication
4. **Local Storage**: All data is stored locally on your device

The app will show a beautiful login screen, followed by baby setup after authentication.

## Features

- ✨ **Premium Claymorphism Design** - Beautiful, soft UI elements
- 📧 **Email Authentication** - No complex OAuth setup needed
- 👶 **Baby Profile Setup** - Easy onboarding experience
